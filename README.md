# Full Stack App
### LinkedIn Learning Project
*MERN stack web development from scratch*

## Project Description
Building a Full Stack application to better understand the differences between Frontend and Backend. 
The LinkedIn project can be found [here](https://www.linkedin.com/learning/react-creating-and-hosting-a-full-stack-site-24928483)

This project was built while completing a MERN‑stack course that pushed my coding and troubleshooting abilities to the limit. Beforehand I understood the front‑end and back‑end concepts in isolation; the course taught me how to wire them together through a MongoDB database, creating a fully functional full‑stack application. The result is a hands‑on demonstration of a seamless connection between React, Express, Node.js, and MongoDB, showcasing both the architecture and the problem‑solving techniques I mastered along the way.


## Why This Project Is on GitHub

I’m publishing this repository to give recruiters and hiring managers concrete evidence of my ability to learn new technologies quickly and apply them end‑to‑end. The code demonstrates:

  * Mastery of the entire MERN stack—from a React front‑end to an Express/Node back‑end wired to MongoDB.
  * Real‑world problem solving: error handling, data validation, asynchronous flow, and performance tuning.
  * Clean, version‑controlled development practices (meaningful commit messages, modular structure, and documentation).
